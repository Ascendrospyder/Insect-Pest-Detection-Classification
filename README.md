# Project Structure

This repository contains all the methods used on AgroPest-12 dataset. Each folder represents a different method used during the project. Every method folder includes its own `README.md` with detailed explanations.

## Folder Overview

### **EDA/**

Contains all Exploratory Data Analysis performed on the dataset

### **Faster-RCNN/**

Includes the implementation, training pipeline, and results for the Faster R-CNN object detection model.

### **ORB_pipeline/**

Contains the ORB (Oriented FAST and Rotated BRIEF) feature detection and matching pipeline.

### **SIFT_KNN_SVM/**

Includes the SIFT feature extraction combined with KNN and SVM classification experiments.

### **SVM_HOG_HNM/**

Contains SVM classifier training using HOG features along with hard negative mining.

### **YOLO_Training Results and Oversampling/**

Includes YOLO model training results, oversampling experiments, and performance analyses.
